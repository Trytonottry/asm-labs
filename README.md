# asm-aes-lab

Учебный проект, демонстрирующий реализацию AES-128 на x86-64 Assembly  
(без AES-NI, полностью вручную: s-box, rcon, key expansion, простой encrypt).

## Возможности

- Key Expansion для AES-128 (176 байт раундовых ключей)
- Простая учебная реализация AES Encrypt (минимизированная для наглядности)
- Демонстрационная программа `demo`
- Makefile для удобной сборки

## Сборка

```bash
make
```
## Запуск

```bash
./demo | xxd
```

Вы увидите 16-байтный зашифрованный блок.

## Структура

```bash
aes128.asm    — учебная реализация AES
main.asm      — пример использования
Makefile      — сборка проекта
```

## Зачем это нужно?
- понять, как устроены блочные шифры изнутри;
- увидеть, как работает S-box и key expansion;
- разобраться, как задаётся roundkey schedule;
- получить опыт low-level криптографии на ASM.

## ⚠️ Это не production AES
Этот код не предназначен для реальной криптографии.
Он создан исключительно для обучения и экспериментов.
